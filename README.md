# 🎓 Tailwind Responsive Grid Dashboard

## 📌 Project Overview

This project demonstrates a **Responsive Grid Layout** using **Tailwind CSS** by building a modern **College Analytics Dashboard**. The dashboard contains six information cards arranged using Tailwind's responsive grid system.

The layout automatically adapts to different screen sizes:

* 📱 Mobile → 1 Column
* 📟 Tablet → 2 Columns
* 💻 Desktop → 3 Columns

This project was developed as part of a **Peer Learning Activity** to understand and demonstrate Tailwind CSS Responsive Grid concepts.

## 📖 Tailwind Classes Used

### Grid Layout

```html
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8">
```

### Explanation

| Class          | Purpose                    |
| -------------- | -------------------------- |
| grid           | Creates grid layout        |
| grid-cols-1    | One column on mobile       |
| md:grid-cols-2 | Two columns on tablets     |
| lg:grid-cols-3 | Three columns on desktops  |
| gap-8          | Adds spacing between cards |

---

## 📱 Responsive Behavior

### Mobile Devices

* 1 Card per Row

### Tablets

* 2 Cards per Row

### Desktop Screens

* 3 Cards per Row

This ensures an optimal user experience across different devices.
